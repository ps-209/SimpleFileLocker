# SimpleFileLocker

<img width="840" height="503" alt="image" src="https://github.com/user-attachments/assets/7f7715b7-570d-411a-a60f-b2d5fda24150" />


Simply file locks with AES-256
1. Select file
2. Put password and choose to lock or unlock the file
3. Set text is invisible(only ntfs system)
4. Start

Warning!
The file hiding feature only works on files within an NTFS system. If you upload this file to the cloud or move it to a USB drive, the data may be lost.
